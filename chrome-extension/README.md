# MyStats Launcher (Chrome Extension)

This is a minimal Chrome extension that opens the MyStats PWA when you click the toolbar icon.

## Local install (Developer Mode)

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the `mystats/chrome-extension/` folder

## Change the target URL

Edit `chrome-extension/background.js`:

- `APP_URL` / `APP_MATCH`
